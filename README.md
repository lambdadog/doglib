# doglib

Single-file public domain libraries for C. Somewhat inspired by
[stb](https://github.com/nothings/stb) but with a much looser focus.

Much of the early parts of this collection of libraries may be focused
on porting over C flavored versions of useful utilities and design
features of [Zig](https://ziglang.org/). This is largely simply
because I prefer the way Zig does certain things.
